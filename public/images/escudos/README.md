# Escudos oficiales de los municipios

Esta carpeta es donde van los escudos heráldicos oficiales de los municipios que aparecen en la sección **"Entidades atendidas"** del CV.

## Cómo añadirlos

1. **Descarga** el escudo oficial del municipio. Puedes buscarlos en:
   - Wikipedia (artículo del municipio → "Escudo")
   - El sitio web oficial de la alcaldía/concejo
   - Wikimedia Commons

2. **Guarda** cada PNG en esta carpeta con estos nombres exactos:

| Municipio | Nombre del archivo |
|---|---|
| Gachancipá | `gachancipa.png` |
| Choachí | `choachi.png` |
| Zipaquirá | `zipaquira.png` |
| Guatavita | `guatavita.png` |
| Zipacón | `zipacon.png` |
| Sopó | `sopo.png` |
| Asocentro Cajicá | `cajica.png` |
| Nemocón | `nemocon.png` |
| El Rosal | `el-rosal.png` |
| Guasca | `guasca.png` |

3. **Listo.** El sitio detecta el archivo automáticamente y reemplaza el escudo placeholder.

## Especificaciones recomendadas

- **Formato**: PNG con fondo transparente (preferido) o blanco
- **Tamaño**: 300×360 px (proporción 5:6, vertical)
- **Peso**: idealmente menos de 100 KB cada uno (puedes optimizar en https://squoosh.app)

## Si el archivo no existe

Si una imagen falla al cargar (no existe el archivo, error 404, etc.), el sitio
usa automáticamente un escudo placeholder con las iniciales del municipio. No
se rompe nada — solo no aparece el escudo oficial hasta que añadas el PNG.

## Nota legal

Los escudos municipales colombianos son símbolos oficiales de uso público.
Asegúrate de respetar las normas locales sobre su reproducción.
